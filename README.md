# Тестовое задание для VERBA-group

## Комментарии:

На данный момент парсер засисит от двух эндпоинов:
- Основной - для получения каталога товаров и большей части информации: https://www.wildberries.ru/__internal/u-search/exactmatch/ru/common/v18/search
- И сервер CDN для получения изображений, описания и характеристик по каждому отдельному товару: https://mow-basket-cdn-01.geobasket.ru/

Также, к сожалению, мне пока не удалость найти способ для автоматической генерации cookies или обхода анти-бот защиты, поэтому в headers захардкожена строка с cookies.

## Использование:

Для того чтобы воспользоваться парсером достаточно передать аргумент query с запросом скрипту parser.py:
```bash
python parser.py --query "пальто из натуральной шерсти"
```
Также можно настроить уровень вывода логов:
- INFO (по умолчанию): Отображает каждый товар, который был добавлен в таблицу в реальном времени
- WARNING: Предупреждения (не используются в скрипте)
- ERROR: Отобрадает только ошибки (проблемы с доступом к серверам, отсутствие или искаженность каких-либо полей и т.д.)
```bash
python parser.py --query "пальто из натуральной шерсти" --log ERROR
```
